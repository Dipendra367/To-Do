# ⚡ TaskFlow — Modern Task Management

TaskFlow is a premium, high-performance to-do application designed for clarity, efficiency, and aesthetics. Built with vanilla technologies, it features a glassmorphic design, advanced filtering, and a robust state management system.



---

## ✨ Key Features

- 🌙 **Adaptive Themes** — Seamless switching between Dark and Light modes with persistent state.
- 📊 **Insight Dashboard** — Real-time statistics for total, pending, completed, and overdue tasks.
- 🏷️ **Categorization** — Organize tasks by Personal, Work, Health, Shopping, Learning, or custom categories.
- 🚦 **Priority System** — Color-coded priority levels (High, Medium, Low) for better focus.
- 📅 **Smart Due Dates** — Automatic overdue detection and "Today" highlighting.
- ✅ **Nested Subtasks** — Break down complex tasks into smaller, manageable steps.
- 🔍 **Live Search & Filter** — Instant searching across titles/notes and status filtering.
- 🖱️ **Drag & Drop** — Intuitive task reordering for optimized workflows.
- 💾 **Local Persistence** — All data is securely stored in your browser's LocalStorage.
- 🎉 **Micro-Animations** — Delightful confetti effects and smooth CSS transitions.

## 🚀 Tech Stack

- **Structure:** Semantic HTML5
- **Logic:** Vanilla JavaScript (ES6+)
- **Styling:** Custom CSS3 with Glassmorphism & Modern Flexbox/Grid
- **Icons:** Unicode/Emojis (Zero-dependency)

## 🛠️ Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dipendra367/test.git
   cd test
   ```

2. **Run locally:**
   Since this is a client-side app, you can simply open `index.html` in your browser. Or use a simple server:
   ```bash
   # Python 3
   python3 -m http.server 5500
   ```

3. **Access the App:**
   Open your browser to `http://localhost:5500`.

## ⌨️ Shortcuts

- `Ctrl` + `/` (or `Cmd` + `/`): Focus the new task input.
- `Enter`: Quickly add a task from the input field.
- `Esc`: Close confirmation modals.
